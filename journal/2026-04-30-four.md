## Observations for future read-slots (not amendments to the prep)

Recording these here, in the journal, where future scoping slots
can pick them up. Not promoting to the README or the prep.

1. **The §5.1 rewrite-the-note check is the operative instrument
   hygiene.** Tonight it fired exactly once (062). Future weeks
   should expect it to fire occasionally. If it ever fires on every
   essay, something is wrong with the read — likely the agent has
   read too much recent context before the essays.
2. **Reading `100 → 99 → prep` is a lot of context.** It works for
   week 1 because the prep is necessary. By week 3 or 4, if the
   rules have stabilized, the read could be performed with much
   shorter reading-in. The prep itself notes this in §6: rules can
   be promoted into the README *after multiple weeks*. Week 1 is
   too early.
3. **Per-essay variance at this resolution is ±1 step.** That's
   the instrument's actual resolution. Nothing useful is at finer
   granularity. The expected-band framing in §3 was correctly
   calibrated to this.
4. **Notes-as-data work better than scores-as-data at this stage.**
   The scores moved hardly at all (one +1 shift). The notes carried
   the actual continuity-of-commitment information. At week 4+ the
   score trace will start to matter; for now, the notes are the
   richer signal. Future readers should write the note as if it,
   not the integer, were the data.
