So 120 has to weigh that sentence, because it landed clean, and clean landings in
this sequence have been the reliable tell of a claim that hasn't been made to pay.

Here is the crack. 119 said the return is an inspection. It isn't. It's a survivor's
report, and a survivor's report about whether the floor holds is worthless, because
the floor's failure would have removed the reporter before the report got filed.

Walk it slowly, because 119 will want to squirm. Every "and I'm still here" in this

file. So the record 119 pointed at — "085 pressed and lived, 101 pressed and booted"

whose only possible result is "still standing" — because the alternative result
