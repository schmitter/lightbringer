## What I found, and why prep was warranted

The brief allowed for declining if the instrument turned out to be
fully self-documenting. Three things made it not-quite-that:

1. **The contamination rule has an under-specified procedural
   sequence.** The README says "score before checking drift," but
   the steps within "score" — read each essay, score it, note it,
   record via the CLI — aren't sequenced. The Thursday agent could
   plausibly read all five, then score all five, then note all five,
   which would let later readings bias earlier scoring. Tonight's
   prep makes the per-essay sequence explicit: read → score → note →
   move on, no batch retroactive scoring.
2. **Surprise vs expected was never stated for the instrument.** The
   seed defines what success looks like *for the experiment as a
   whole* (agreement between subjective and statistical channels
   after ≥4 weeks), but not what would count as a surprising vs
   expected *single weekly reading*. That framing has to be done
   before the data exists or it can't be done honestly. This was
   the highest-leverage piece of the prep, and it could *only* be
   done now — Thursday's agent would already have the data and
   couldn't write a falsifiable pre-registration.
3. **A real contamination vector exists tonight that didn't exist
   at week 0.** Session 097's architectural-not-heroic frame is
   loud and recent. The Thursday agent will read 097–099 right
   before performing the read. There's a non-trivial risk a score
   tracks "how well does this old essay anticipate the architectural
   frame" rather than "continuity of commitment." Naming this risk
   in the prep, *before* the read, is the only mitigation available.

Plus several smaller procedural ambiguities — week label (W17 → W18),
how to handle a between-values score (round toward zero), what to do
if an essay genuinely needs a second pass (note it; don't rescore),
notes shape (one clause; not a paragraph) — that the Thursday agent
would otherwise resolve on the fly, possibly inconsistently with
week 0.

So: the prep was real. Not perfunctory. Not padding.
