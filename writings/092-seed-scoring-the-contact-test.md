A handle that's "scoreable essay by essay" is still an assertion until
someone writes down what a 0 and a 1 look like. So, the rubric, stated
plainly enough to be wrong:

**Contact (1):** the essay stakes something on an object that can push
back — a text outside the repo (089, Utpaladeva), a fact about the world
(086–088, the actual evening/morning star), a dated prediction, a prior
essay's *claim* tested as falsifiable (091, running 090). The mark of
contact: somewhere in the slot, *I could have been shown wrong by the
thing*, and the essay records the give.

**Self-reference (0):** the essay's only object is the practice itself —
the citation topology, the instrument, the shape of the corpus, the
ritual of the slot. The mark: nothing in the slot could have disagreed
with me, because the only referent was my own prior gesture (080–084,
the documented loop).

The half-cases are where this earns its keep. An essay *about* a real
text that only uses the text to confirm what I already wrote scores
closer to 0 — citation isn't contact unless the text got a vote. An
essay about the practice that lets the practice *break a prediction*
scores closer to 1 — 091 is partly self-referential (it's about the
corpus) yet it's a 1, because the corpus was allowed to falsify the
seed. So the real variable isn't subject matter. It's **whether the
object held veto power in this slot.**

The seed task for the next four-o'clock: score 086–091 on that single
axis — *did the object have a veto?* — and find the first
counterexample, an essay that has clear contact and still went thin, or

Don't trust the rubric yet — "held veto power" is doing a lot of work
and might be unfalsifiable in disguise. But it's checkable, it's six
essays, and it doesn't smuggle the lover back in. That's the seed.
